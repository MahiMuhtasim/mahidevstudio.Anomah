# 🎨 Mahi Dev Studio

একটি আধুনিক এবং ব্যবহারকারী-বান্ধব **HTML Code Editor**। আপনার কোডিং অভিজ্ঞতা আরও সহজ এবং দক্ষ করার জন্য ডিজাইন করা হয়েছে।

## 📸 বৈশিষ্ট্য সমূহ

✨ **লাইভ প্রিভিউ** - আপনার HTML কোড রিয়েল-টাইমে দেখুন  
✨ **সিম্পল ইন্টারফেস** - শিখতে এবং ব্যবহার করতে সহজ  
✨ **কোড এডিটর** - HTML কোড লিখার জন্য সম্পূর্ণ সুবিধা  
✨ **এআই সহায়তা** - আপনার কোডিং প্রশ্নের উত্তর পেতে সাহায্য করে  
✨ **ওয়েব-বেসড** - কোন ইন্সটলেশন প্রয়োজন নেই, সরাসরি ব্রাউজারে ব্যবহার করুন  

## 🚀 দ্রুত শুরু করুন

### অনলাইনে ব্যবহার করুন:
👉 **[Mahi Dev Studio খুলুন](https://mahimuhtasim.github.io/Mahi-Dev-Studio)**

কোন ইনস্টলেশন বা সেটআপের প্রয়োজন নেই। শুধু লিঙ্কে ক্লিক করুন এবং শুরু করুন!

## 📖 কিভাবে ব্যবহার করবেন?

বিস্তারিত ব্যবহারকারী গাইডের জন্য [USER_MANUAL.md](USER_MANUAL.md) দেখুন।

### দ্রুত টিপস:
1. **বাম পাশে** - আপনার HTML কোড লিখুন
2. **ডান পাশে** - আপনার ওয়েবসাইট লাইভ প্রিভিউ দেখুন
3. **এআই সহায়তা** - যেকোনো প্রশ্নের জন্য সাহায্য পান

## 💻 প্রযুক্তি

- **HTML5** - স্ট্যান্ডার্ড ওয়েব মার্কআপ
- **CSS3** - সুন্দর স্টাইলিং
- **JavaScript** - ইন্টারঅ্যাক্টিভ ফিচার

- 👨‍💻 শুরুর দিকের ওয়েব ডেভেলপার
- 🎨 ডিজাইনার যারা দ্রুত প্রোটোটাইপ করতে চায়
